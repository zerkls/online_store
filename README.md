# Интернет-магазин - WPF приложение

## О проекте

Курсовой проект по дисциплине **"Объектно-ориентированное программирование"**. 
Полнофункциональная информационная система для интернет-магазина, разработанная с использованием WPF и паттерна MVVM.

Проект демонстрирует применение основных принципов объектно-ориентированного программирования и современных подходов к разработке desktop-приложений.

## Возможности

### Основной функционал
- **Каталог товаров** - просмотр и поиск товаров
- **Умный поиск** - фильтрация по названию и категориям
- **Корзина покупок** - добавление, удаление, изменение количества товаров
- **Оформление заказов** - полный цикл от выбора товаров до подтверждения
- **История заказов** - просмотр и управление предыдущими заказами
- **Система оплаты** - поддержка различных способов оплаты

## Архитектура

### Технологический стек
- **Язык программирования**: C# 9.0
- **Платформа**: .NET 6.0
- **UI фреймворк**: WPF (Windows Presentation Foundation)
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Хранение данных**: In-memory (в памяти приложения)
